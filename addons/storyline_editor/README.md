# Storyline Graph Editor

Editor plugin that visualizes and edits storyline event dependencies (route browser, graph,
and validation/inspector bridge). It reads and writes the canonical storyline route resources
without introducing a separate source of truth. Enable via Project → Project Settings → Plugins.

This README is the plugin's entry point; the full documentation lives in [`docs/`](docs):

- [`docs/feature.md`](docs/feature.md) — goals, the three panels, rules, edge cases, ownership, and validation.
- [`docs/contract.md`](docs/contract.md) — panel registration, signals, refresh path, and canonical-source contract.

The project-level [`../../docs/module_map.md`](../../docs/module_map.md) and
[`../../docs/architecture.md`](../../docs/architecture.md) keep one-line pointers here.
